Puzzle de 3 x 3

Este proyecto implementa el Puzzle de 8, un rompecabezas 3x3 resuelto mediante el algoritmo de búsqueda A*. La aplicación interactiva permite al usuario definir estados inicial y meta, validando su resolubilidad. Utiliza la heurística de distancia de Manhattan para encontrar la solución óptima. Demuestra la aplicación de IA para resolver problemas complejos de manera eficiente.
